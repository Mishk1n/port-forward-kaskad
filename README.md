# Port Forwarding Manager (kaskad)
Скрипт для быстрого создания «каскадного» перенаправления портов через VPS с помощью iptables.

## Возможности
- Проброс **UDP** (WireGuard/AmneziaWG) и **TCP** (VLESS/XRay, MTProto)
- **Кастомные правила** с разными входящим/исходящим портами
- Просмотр и удаление правил
- Полный сброс настроек
- Встроенная инструкция по настройке клиента

## Установка и запуск
```bash
# Сделайть исполняемым
chmod +x port-forward-kaskad.sh
# Запустите от root
sudo ./port-forward-kaskad.sh
```
После первого запуска скрипт сам скопирует себя в /usr/local/bin/kaskad — в дальнейшем можно вызывать командой sudo kaskad.

## Быстрый старт
1. Выберите тип правила (1–3 для готовых или 4 для кастомного)
2. Введите IP и порты зарубежного сервера (VPN/прокси)
3. На клиенте замените адрес сервера на IP этого VPS
4. Готово: Клиент → Ваш VPS → Зарубежный сервер

## Зависимости
Устанавливаются автоматически:
> iptables-persistent
netfilter-persistent

Включается IP forwarding и BBR.

## Меню
| Пункт | Описание                                          |
|-------|---------------------------------------------------|
| 1     | Настройка AmneziaWG / WireGuard (UDP)            |
| 2     | Настройка VLESS / XRay (TCP)                     |
| 3     | Настройка MTProto / TProxy (TCP)                 |
| 4     | Кастомное правило (разные порты, SSH, RDP и др.) |
| 5     | Показать активные правила                        |
| 6     | Удалить одно правило                             |
| 7     | Сбросить ВСЕ настройки iptables                  |
| 8     | Инструкция по настройке клиента                  |
| 0     | Выход                                            |

## Важно
- Запускать только от root
- Для сохранения правил после перезагрузки используется netfilter-persistent save
- Если активен UFW, скрипт автоматически добавит исключения

## Пример кастомного правила
```
Протокол: tcp
IP назначения: 123.123.123.123
Входящий порт: 4443
Исходящий порт: 443
```

→ Клиент подключается к вашему VPS:4443, трафик уходит на 123.123.123.123:443
